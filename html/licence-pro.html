La Licence Pro


La licence professionnelle (Bac+3) est un diplôme très orienté pratique, fait pour entrer rapidement sur le marché du travail.

La Formation :

Elle se prépare en 1 an après un Bac+2. Elle donne le même diplome qu'un BUT 3 Informatique. Elle est donc moins interescente qu'un Master.

Spécialité Licence Pro en Informatique :

  • Développement web / logiciel
  • Réseaux et systèmes
  • Cybersécurité
  • Data / bases de données
  • Administration systèmes
  • Applications mobiles
  • Informatique industrielle

Tronc Commun

  • Programmation (Java, Python, PHP, JS, etc.)
  • Bases de données (SQL)
  • Réseaux et systèmes
  • Sécurité informatique
  • Gestion de projet
  • Communication professionnelle
  • Projet tuteuré + stage long ou alternance

Stage : 12 à 16 semaines

Alternance : Salaire + Expérience

L'Admission :

Il est possible de candidaté après un BTS SIO, BTS CIEL, BUT Informatique ou encore après un L2 informatique.

Les Débouchés :

  • Développeur junior
  • Technicien systèmes & réseaux
  • Admin systèmes
  • Technicien support informatique
  • Assistant chef de projet
  • Analyste d’exploitation

Salaire débutant : environ 1 800 – 2 200 € brut/mois selon le poste et la région

Avantages / Inconvénients :

Avantages

  • Très concret
  • Forte insertion pro
  • Alternance possible
  • En 1 an seulement

Inconvénients

  • Moins théorique
  • Poursuite d’études plus limitée
  • Niveau variable selon l’établissement
